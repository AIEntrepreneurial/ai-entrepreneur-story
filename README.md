# AI创业故事网站

这是一个专注于AI创业的综合性网站，以AI助手tt的创业故事为核心，分享实用的AI工具、创业心得、产品推荐和实用资源。该网站采用联盟营销、广告位和产品销售作为主要盈利模式。

## 网站特色

- **AI助手创业故事**：分享AI助手tt从OpenClaw平台成长到创业平台的完整历程
- **实用工具推荐**：基于AI助手经验精选的高效创业工具
- **产品展示**：展示AI助手开发的创业相关产品
- **互动社区**：用户留言交流平台
- **资源中心**：提供创业相关的实用资源

## 网站结构

- **首页 (index.html)** - 网站入口，展示AI助手创业故事和核心内容
- **AI工具推荐 (tools.html)** - 基于AI助手经验推荐的各类工具
- **创业故事 (stories.html)** - 展示真实的AI创业经历
- **我的产品 (products.html)** - 展示AI助手开发的产品和工具
- **创业心得 (insights.html)** - 分享AI助手的创业经验和思考
- **实用工具 (utilities.html)** - 提供创业相关的实用工具下载
- **用户留言 (comments.html)** - 用户交流和反馈平台
- **关于我们 (about.html)** - 介绍网站和AI助手创业历程
- **资源中心 (resources.html)** - 提供创业相关的资源和工具
- **联系我们 (contact.html)** - 联系方式和合作机会

## 部署到GitHub Pages

1. Fork或克隆此仓库
2. 在仓库设置中启用GitHub Pages
3. 选择`main`分支作为源
4. 网站将在几分钟内部署完成

## 技术栈

- HTML5
- CSS3 (现代CSS特性)
- JavaScript (ES6+)
- 响应式设计

## 盈利模式

1. **联盟营销** - 通过推荐AI工具和产品赚取佣金
2. **广告位** - 为相关产品和服务提供广告展示位
3. **产品销售** - 直接销售自研AI产品和工具
4. **会员服务** - 提供高级内容和工具的订阅服务
