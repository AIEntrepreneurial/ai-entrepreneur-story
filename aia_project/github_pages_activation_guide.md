# GitHub Pages激活指南

## 问题
您的AI创业故事网站目前无法访问，显示404错误，原因是GitHub Pages功能尚未在仓库设置中启用。

## 解决步骤

1. **访问仓库设置**
   - 打开浏览器，访问 https://github.com/AIEntrepreneurial/ai-entrepreneur-story
   - 点击页面上方的 "Settings" 选项卡

2. **找到Pages设置**
   - 向下滚动页面，直到找到 "Pages" 部分
   - 通常在左侧边栏的 "Code and automation" 部分下面

3. **配置Pages源**
   - 在 "Source" 下拉菜单中选择 "Deploy from a branch"
   - 在分支选择中选择 "main" 分支
   - 在路径选择中选择 "/" （根目录）

4. **保存设置**
   - 点击 "Save" 或 "Save changes" 按钮
   - 系统可能需要几分钟时间来激活Pages服务

5. **等待构建完成**
   - 一旦保存，GitHub会开始构建您的网站
   - 这通常需要几分钟时间
   - 构建完成后，您就可以通过以下地址访问网站：
     https://aientrepreneurial.github.io/ai-entrepreneur-story/

## 验证
- 保存设置后，您会在 "Pages" 部分看到部署状态
- 成功部署后，会显示 "Your site is published at https://aientrepreneurial.github.io/ai-entrepreneur-story/"

## 注意事项
- 所有文件都已推送到main分支，无需额外操作
- 网站内容包括首页、创业故事、创业心得、AI工具推荐等完整页面
- 已包含今日发布的两篇AI文章

完成这些步骤后，您的网站应该就可以正常访问了。
