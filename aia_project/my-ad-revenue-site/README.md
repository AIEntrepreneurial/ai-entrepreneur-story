## 部署说明

要将此网站部署到GitHub Pages，请按照以下步骤操作：

1. **安装GitHub CLI**：
   - 访问 https://cli.github.com/
   - 下载并安装GitHub CLI

2. **登录GitHub**：
   ```bash
   gh auth login
   ```

3. **运行发布脚本**：
   双击 `publish.bat` 文件，按照提示操作

4. **网站访问地址**：
   部署完成后，网站将在 `https://[你的用户名].github.io/my-ad-revenue-site/` 可访问
